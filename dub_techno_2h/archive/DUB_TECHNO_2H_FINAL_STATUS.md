## NEXT STEPS TO COMPLETE

The **foundation is 100% complete**. To finish the track:

### 1. Configure Instruments (in Ableton)

**Time Required**: 10-15 minutes

- Open each instrument device
- Choose appropriate presets or design sounds
- **Kick**: Design punchy kick or use preset
- **Sub-bass**: Design deep sub sound
- **Hi-hats**: Load hi-hat samples in Drum Rack
- **Synth Pads**: Choose atmospheric pad presets
- **FX**: Load FX samples (sweeps, impacts)

Reference: **`DUB_TECHNO_2H_COMPLETE_GUIDE.md`** for detailed settings

### 2. Set Up Send Routing (in Ableton)

**Time Required**: 5 minutes

For each main track (4-9):
1. Expand mixer section (click track header)
2. Click "Sends" button
3. Set Send A (Reverb) and Send B (Delay) levels:

| Track | Reverb Send | Delay Send |
|-------|-------------|------------|
| 4 - Kick | 0% | 0% |
| 5 - Sub-bass | 0% | 5-10% |
| 6 - Hi-hats | 20-30% | 25-35% |
| 7 - Synth Pads | 40-50% | 20-30% |
| 8 - FX | 60-70% | 70-80% |
| 9 - Dub Delays | 80-90% | 100% |

### 3. Mix Track Levels (in Ableton)

**Time Required**: 10 minutes

Set target levels:

| Track | Target Level | Role |
|-------|-------------|------|
| 4 - Kick | -6 to -8 dB | Reference point |
| 5 - Sub-bass | -3 to -6 dB | Deep power |
| 6 - Hi-hats | -12 to -15 dB | Subtle rhythm |
| 7 - Synth Pads | -18 to -24 dB | Background atmosphere |
| 8 - FX | -15 to -20 dB | Accents |
| 9 - Dub Delays | -15 to -18 dB | Echo returns |
| 10 - Reverb Send | -15 dB | Reverb level |
| 11 - Delay Send | -12 dB | Delay level |

### 4. Configure Effects (in Ableton)

**Time Required**: 10 minutes

**Hybrid Reverb (Track 10)**:
- Size: Large (70-90%)
- Decay Time: 3-5 seconds
- Diffusion: 80-90%
- Volume: -15 dB

**Simple Delay (Track 11)**:
- Time: 1/4 or 1/8 notes
- Feedback: 35-50%
- Dry/Wet: 30-40% wet
- Volume: -12 dB

**Auto Filter (Track 7)**:
- Filter Type: Low 24 dB
- Frequency: 400-2000 Hz (automate this!)
- Resonance: 0.3-0.5

### 5. Add Automation (in Ableton)

**Time Required**: 20-30 minutes

**Filter Automation (Track 7)**:
- Breakdowns: 400-600 Hz (closed)
- Buildups: Sweep up to 2000 Hz
- Peaks: 1500-2000 Hz (open)
- Returns: Sweep down to 800 Hz

**Send Automation (Reverb/Delay)**:
- Breakdowns: Increase to 50-60%
- Peaks: Decrease to 20-30%
- Use smooth curves

### 6. Test Playback

**Time Required**: 2 hours (full listen!)

1. Start playback from Section 0
2. Listen through all 30 sections
3. Check transitions (every 4 minutes)
4. Adjust mix levels as needed
5. Refine automation
