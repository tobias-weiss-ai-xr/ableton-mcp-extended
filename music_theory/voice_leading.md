# Voice Leading Reference

> Techniques for smooth chord transitions, inversions, and bass motion

---

## What Is Voice Leading?

**Voice leading** is the art of connecting chords smoothly by minimizing the distance each voice (note) moves between chords.

### The Golden Rule

**Move each voice as little as possible.**

```
Bad voice leading:
Cmaj7 → Fmaj7
C-E-G-B → F-A-C-E (lots of jumping)

Good voice leading:
Cmaj7 → Fmaj7/C
C-E-G-B → C-F-A-E (minimal movement)
```

## Voice Leading Principles

### Principle 1: Common Tones

**Keep notes that are shared between chords.**

```
Cmaj7 (C-E-G-B) → Em7 (E-G-B-D)
Common tones: E, G, B (keep these!)
Only C moves to D
```

### Principle 2: Stepwise Motion

**Move voices by step (whole or half) rather than leap.**

```
Bad:   C → F (4th leap)
Good:  C → D (step)
       D → E (step)
       E → F (step)
```

### Principle 3: Contrary Motion

**When one voice goes up, another goes down.**

```
Parallel motion (avoid):
Top voice:    C → D → E
Bottom voice: G → A → B (both going up)

Contrary motion (prefer):
Top voice:    C → D → E
Bottom voice: G → F → E (opposite direction)
```

### Principle 4: Avoid Parallel Fifths and Octaves

**In classical harmony, avoid parallel 5ths and 8ves between outer voices.**

```
Parallel 5ths (avoid):
C-G → D-A (both voices move same direction to 5th)

Parallel 8ves (avoid):
C-C → D-D (both voices double at octave)
```

*Note: In electronic/pop music, this rule is often broken intentionally.*

## Inversions for Smooth Voice Leading

### Why Use Inversions?

Inversions allow you to keep the **bass note** close between chord changes.

### Example: I-IV Progression

```
Root position only:
Cmaj7 → Fmaj7
C bass → F bass (leap of 4th)

Using inversions:
Cmaj7 → Fmaj7/C
C bass → C bass (no movement!)
```

### Common Inversion Patterns

| Progression | Root Position | With Inversions | Improvement |
|-------------|---------------|-----------------|-------------|
| I → IV | C → F | C → F/C | Bass stays on C |
| I → V | C → G | C → G/B | Bass moves C→B (step) |
| I → vi | C → Am | C → Am/C | Bass stays on C |
| V → I | G → C | G → C/G | Bass stays on G |

## Bass Line Voice Leading

### Types of Bass Motion

| Type | Description | Character |
|------|-------------|-----------|
| **Static** | Same note | Stable, grounded |
| **Stepwise** | Scale motion | Smooth, flowing |
| **Arpeggiated** | Chord tones | Classical, melodic |
| **Leaping** | Large intervals | Energetic, dramatic |
| **Chromatic** | Half-step motion | Jazz, sophisticated |

### Bass Line Patterns

#### Static Bass (Pedal Point)

```
Cmaj7 → Am7 → Fmaj7 → G7
C bass → C bass → C bass → C bass
(All chords over C pedal)
```

**Use for:** Building tension, drone effects

#### Stepwise Bass

```
Cmaj7 → Dm7 → Em7 → Fmaj7
C bass → D bass → E bass → F bass
(Ascending scale motion)
```

**Use for:** Smooth progressions, ascending energy

#### Chromatic Bass

```
Cmaj7 → C#dim7 → Dm7 → D#dim7 → Em7
C → C# → D → D# → E
(Chromatic ascent)
```

**Use for:** Jazz, sophisticated progressions

#### Circle of Fifths Bass

```
Cmaj7 → Bm7b5 → E7 → Am7 → Dm7 → G7 → Cmaj7
C → B → E → A → D → G → C
(Descending 5ths)
```

**Use for:** Jazz, classical, continuous motion

## Four-Part Voice Leading

### SATB (Soprano-Alto-Tenor-Bass)

Traditional four-part writing assigns notes to voice parts:

```
Soprano: Highest voice (melody)
Alto:    Upper middle
Tenor:   Lower middle
Bass:    Lowest voice (foundation)
```

### Spacing Rules

| Rule | Description |
|------|-------------|
| **Close spacing** | All voices within one octave (except bass) |
| **Open spacing** | Voices spread across multiple octaves |
| **No crossing** | Soprano > Alto > Tenor > Bass |

### Example: Cmaj7 to Fmaj7

```
Close spacing:
Soprano: B → C
Alto:    G → A
Tenor:   E → F
Bass:    C → F

Open spacing:
Soprano: B → C
Alto:    E → F
Tenor:   G → A
Bass:    C → F
```

## Jazz Voice Leading

### Shell Voicings (3-7)

Jazz pianists often play only the 3rd and 7th of chords:

```
Cmaj7: E-B (3rd and 7th)
Dm7:   F-C (3rd and 7th)
G7:    B-F (3rd and 7th)
Cmaj7: E-B (3rd and 7th)

Notice: E→F→B→E (minimal movement)
        B→C→F→B (minimal movement)
```

### Guide Tones

The **3rd and 7th** are guide tones - they define the chord quality:

| Chord | 3rd | 7th | Quality |
|-------|-----|-----|---------|
| Cmaj7 | E | B | Major 7 |
| C7 | E | Bb | Dominant |
| Cm7 | Eb | Bb | Minor 7 |

### ii-V-I Voice Leading

```
Dm7    →   G7    →   Cmaj7
F-C        B-F       E-B
(3-7)      (7-3)     (3-7)

Notice:
F → F (stays)
C → B (half step)
B → B (stays)
F → E (half step)
```

## Keyboard Voicings

### Rootless Voicings

When a bass player is present, omit the root:

```
Full Cmaj7:    C-E-G-B
Rootless:      E-G-B (or E-B-D for 9th sound)
```

### Drop 2 Voicings

Drop the 2nd note from top down an octave:

```
Close:   C-E-G-B
Drop 2:  G-C-E-B (G dropped down)

Close:   D-F-A-C (Dm7)
Drop 2:  A-D-F-C (A dropped down)
```

### So What Voicings (4th Chords)

Stack perfect 4ths for modal jazz sound:

```
Dm11: D-G-C-F (all 4ths)
Em11: E-A-D-G (all 4ths)
```

## Common Voice Leading Patterns

### Pattern 1: Descending Soprano

```
Cmaj7 → Cmaj7/E → F → F/G
G     → G     → A → G
(soprano descends: G-G-A-G doesn't work well)

Better:
Cmaj7 → Cmaj7/E → F → G7
G     → G     → A → G
E     → E     → F → F
C     → E     → C → G
```

### Pattern 2: Ascending Bass

```
C → C#dim7 → Dm7 → D#dim7 → Em7
Smooth chromatic ascent in bass
```

### Pattern 3: 7-3 Resolution

```
G7    → Cmaj7
B (7) → C (3)
Leading tone resolves up to tonic
```

### Pattern 4: 4-3 Suspension

```
G7sus4 → G7 → Cmaj7
C (4)  → B (3) → C
Suspension resolves down, then chord changes
```

## Electronic Music Voice Leading

### Pad Voice Leading

Electronic pads often use:

1. **Common tone retention** - Keep one note across chords
2. **Slow movement** - Pads move gradually
3. **Parallel motion** - Intentional parallel chords

```
Am7 → Fmaj7 → C → G
A-C-E-G → A-C-F-A → C-E-G-C → B-D-G-B
(Keeping A as common tone)
```

### Synth Chord Stabs

For staccato chords:

```
Close voicings (tight sound)
Parallel motion (unified movement)
Limited range (punchy)
```

### Dub Techno Chords

```
Am7 → Gm7
A-E-G-C → G-D-F-Bb
Wide voicings, sparse notes
Heavy delay/reverb masks voice leading
```

## Practice Exercises

### Exercise 1: Identify Voice Leading Issues

Find the voice leading problems:

```
Cmaj7 → G7 → Cmaj7
C-E-G-B → G-B-D-F → C-E-G-B
```

<details>
<summary>Answer</summary>
- Bass leaps C→G→C (acceptable but could be smoother)
- Soprano: B→B→B (static, no movement)
- Parallel 5ths between tenor and bass in second chord
</details>

### Exercise 2: Improve Voice Leading

Rewrite with better voice leading:

```
Cmaj7 → Fmaj7
C-E-G-B → F-A-C-E
```

<details>
<summary>Answer</summary>
```
Cmaj7 → Fmaj7/C
C-E-G-B → C-F-A-E

- Bass stays on C
- E→F (half step)
- G→A (whole step)
- B→E (down a 4th, or keep B and add E)
```
</details>

### Exercise 3: Write a Bass Line

Write a bass line for: Dm7 → G7 → Cmaj7

<details>
<summary>Suggested Answer</summary>
```
D → D → G → G → C → C
(stepwise in Dm7, then to G, then to C)

Or chromatic:
D → D# → E → F → F# → G → G# → A → Bb → B → C
```
</details>
