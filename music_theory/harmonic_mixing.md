## Mixing Rules

### Rule 1: Perfect Mix (Same Key)

**Change:** 0 semitones

**Transition:** 8A → 8A, 8B → 8B

**Effect:** Seamless, no harmonic tension

**Best For:**
- Long blends
- Loop-based mixing
- Same-energy transitions

**Example:**
```
Current: Am (8A)
Next: Am (8A)
Result: Perfect match, seamless blend
```

### Rule 2: Energy Up (+1)

**Change:** +1 semitone (half step up)

**Transition:** 8A → 9A, 8B → 9B

**Effect:** Slight energy boost, brighter

**Best For:**
- Building energy
- Peak-time transitions
- Uplifting moments

**Example:**
```
Current: Am (8A)
Next: Em (9A)
Result: Energy lift, brighter feel
```

### Rule 3: Energy Down (-1)

**Change:** -1 semitone (half step down)

**Transition:** 8A → 7A, 8B → 7B

**Effect:** Slight energy drop, deeper

**Best For:**
- Winding down
- Deeper vibes
- Transitioning to darker tracks

**Example:**
```
Current: Am (8A)
Next: Dm (7A)
Result: Energy drop, deeper feel
```

### Rule 4: Relative Major/Minor

**Change:** Switch A ↔ B (same number)

**Transition:** 8A → 8B, 8B → 8A

**Effect:** Mood shift without key change

**Best For:**
- Minor to major (uplifting)
- Major to minor (darker)
- Emotional transitions

**Example:**
```
Current: Am (8A)
Next: C (8B)
Result: Minor to major = brighter, more uplifting
```

### Rule 5: Two-Step Energy (+2 or -2)

**Change:** ±2 semitones

**Transition:** 8A → 10A, 8A → 6A

**Effect:** Noticeable energy shift

**Best For:**
- Dramatic transitions
- Genre switches
- Peak → breakdown or vice versa

**Example:**
```
Current: Am (8A)
Next: Bm (10A)
Result: Noticeable energy boost
```

### Rule 6: The "Energizer" (+7)

**Change:** +7 semitones (perfect 5th up)

**Transition:** 8A → 3A (wraps around)

**Effect:** Significant energy boost

**Best For:**
- Big drops
- Peak-time moments
- Waking up the crowd

**Example:**
```
Current: Am (8A)
Next: C#m (3A)
Result: Big energy boost
```

## Transition Matrix

### Compatible Keys (From 8A - A Minor)

| Target | Transition | Semitones | Energy Change | Risk |
|--------|------------|-----------|---------------|------|
| 8A | Same key | 0 | None | Safe |
| 8B | Relative major | 3 | Brighter | Safe |
| 9A | Energy up | +1 | Slight boost | Safe |
| 7A | Energy down | -1 | Slight drop | Safe |
| 10A | Two-step up | +2 | Moderate boost | Medium |
| 6A | Two-step down | -2 | Moderate drop | Medium |
| 3A | Energizer (+7) | +7 | Big boost | Bold |
| 5A | Subdominant | +5 | Interesting | Bold |

### Incompatible Keys (Avoid)

| Target | From 8A | Semitones | Issue |
|--------|---------|-----------|-------|
| 2A | +7 (tritone) | 6 | Dissonant, jarring |
| 4A | +9 | 9 | Clashes |
| 11A | +4 | 4 | Awkward |
| 1A | +10 | 10 | Harsh |

### Energy-Based Key Selection

| Set Phase | Energy | Recommended Transitions | Example |
|-----------|--------|------------------------|---------|
| **Warmup** | Low | Same, -1, Relative minor | 8A → 7A → 8A |

## Genre-Specific Mixing

### House / Deep House

**Typical Keys:** Minor keys (A) dominant

**Common Transitions:**
- 8A → 9A → 10A (gradual energy up)
- 8A → 8B (minor to major = uplifting)

**Example Set Flow:**
```
8A (Am) → 9A (Em) → 9A (Em) → 10A (Bm) → 10B (D) → 10A (Bm) → 9A (Em)
```

### Techno

**Typical Keys:** Minor keys, darker scales

**Common Transitions:**
- Same key for long blends
- -1 for deeper sections
- Relative major for melodic moments

**Example Set Flow:**
```
8A (Am) → 8A (Am) → 7A (Dm) → 7A (Dm) → 6A (Gm) → 6A (Gm) → 7A (Dm)
```

### Dub Techno

**Typical Keys:** Minor, atmospheric

**Common Transitions:**
- Extended same-key mixing
- Subtle energy changes (-1, +1)
- Rarely use major keys

**Example Set Flow:**
```
8A (Am) → 8A (Am) → 8A (Am) → 9A (Em) → 9A (Em) → 8A (Am)
```

### Trance

**Typical Keys:** Major keys (B) for uplifting feel

**Common Transitions:**
- 8B → 9B → 10B (energy up)
- Relative minor for breakdown
- +2 for big drops

**Example Set Flow:**
```
8B (C) → 9B (G) → 9B (G) → 10B (D) → 10A (Bm) → 10B (D) → 11B (A)
```

### Drum & Bass

**Typical Keys:** Minor keys, often darker

**Common Transitions:**
- Same key for consistency
- -1 for deeper sections
- Relative major for melodic drops

**Example Set Flow:**
```
8A (Am) → 8A (Am) → 7A (Dm) → 7A (Dm) → 8A (Am) → 9A (Em)
```

## BPM Considerations

### Key Pitch Changes with Tempo

When you change tempo, the key shifts:

| BPM Change | Semitone Shift | Key Change |
|------------|----------------|------------|
| +6% | +1 semitone | 8A → 9A |
| -6% | -1 semitone | 8A → 7A |
| +12% | +2 semitones | 8A → 10A |
| -12% | -2 semitones | 8A → 6A |

### Master Tempo / Key Lock

**When Enabled:**
- Key stays constant regardless of BPM
- Use Camelot wheel normally

**When Disabled:**
- Key shifts with BPM
- Account for pitch change in key selection

### Example: Key Lock OFF

```
Track 1: 126 BPM, Am (8A)
Track 2: 130 BPM, Gm (7A)

If you pitch Track 2 up to match 130 BPM:
- +3.2% = +0.5 semitones (approximately)
- Gm becomes G#m (between 7A and 8A)

Solution: Use key lock OR select tracks closer in BPM
```

## Advanced Techniques

### Technique 1: The "Mini-Modulation"

**Concept:** Quick key change for energy

```
8A → 8A → 10A → 8A
(Am → Am → Bm → Am)
```

**Effect:** Brief energy boost, return home

### Technique 2: The "Major Lift"

**Concept:** Minor to relative major for breakdown

```
8A → 8A → 8B → 8A
(Am → Am → C → Am)
```

**Effect:** Uplifting breakdown, darker drop

### Technique 3: The "Tritone Tension"

**Concept:** Use tritone for tension (risky!)

```
8A → 2A → 8A
(Am → G#m → Am)
```

**Effect:** Creates tension, resolves home

**Warning:** High risk, use sparingly!

### Technique 4: The "Circle of Fifths Climb"

**Concept:** Ascend through circle of fifths

```
8A → 3A → 10A → 5A → 12A
(Am → C#m → Bm → Cm → D#m)
```

**Effect:** Continuous energy rise

### Technique 5: The "Double Drop"

**Concept:** Mix two tracks in same key

```
Track 1: 8A (Am)
Track 2: 8A (Am)
Mix: Both play together = 8A
```

**Effect:** Massive sound, thick texture

## Practical Workflow

### Step 1: Analyze Your Library

Use key detection software:
- Mixed In Key
- rekordbox
- Serato
- Traktor
- Keyfinder

Tag files with Camelot notation.

### Step 2: Organize by Key

Create playlists:
- `8A - A Minor`
- `8B - C Major`
- `9A - E Minor`
- etc.

### Step 3: Plan Transitions

Before set:
1. List tracks in order
2. Check key compatibility
3. Plan energy arc
4. Identify risk points

### Step 4: Live Mixing

During set:
1. Know current key
2. Filter next track by compatible keys
3. Check BPM compatibility
4. Execute transition

## Common Mistakes

### Mistake 1: Ignoring Key Completely

**Problem:** Random key changes sound jarring

**Solution:** Always check key before mixing

### Mistake 2: Over-Planning

**Problem:** Rigid set plan limits creativity

**Solution:** Plan key zones, stay flexible within

### Mistake 3: Same Key Too Long

**Problem:** Monotonous, no progression

**Solution:** Change key every 3-5 tracks

### Mistake 4: Ignoring Energy Arc

**Problem:** Flat energy throughout set

**Solution:** Map energy arc, use key to support

### Mistake 5: Forgetting BPM Affects Key

**Problem:** Key lock off, pitch changes key

**Solution:** Use key lock OR account for pitch shift
